# FinanceOS V1.6 Clean Core

بازنویسی کامل منطق داده و ارتباط ماژول‌ها.

- حذف کامل منطق قسط پیش‌فرض
- حساب‌ها از موجودی اولیه + تراکنش‌ها محاسبه می‌شوند
- اقساط فقط از لیست اقساط و بدهی‌ها محاسبه می‌شود
- پرداخت قسط فقط یک تراکنش است و از حساب مبدا کم می‌شود
- داشبورد فقط از داده‌های واقعی محاسبه می‌شود
- ریست تمیز برای پاک‌کردن داده‌های نسخه‌های قبلی
