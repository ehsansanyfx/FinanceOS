# Product Plan — FinanceOS

## Current Version

V1.3.0

## Product Positioning

FinanceOS is a Persian-first Personal Financial Command Center for people who want a structured way to manage income, accounts, installments, expenses, assets, liabilities, goals, emergency funds, and financial health.

## Core Problem

Most people only track expenses after money is already spent.  
FinanceOS changes the flow:

Income → Allocate → Separate Accounts → Track Transactions → Build Assets → Reduce Liabilities → Fund Goals → Analyze → Correct
